Inputting missing values

New dataset that is equal to the original dataset but with the missing data filled

activity2 <- activity
for (i in 1:nrow(activity)){
    if(is.na(activity$steps[i])){
        activity2$steps[i]<- Avg_Steps_alldays[[as.character(activity[i, "interval"])]]
        echo=TRUE
    }
}

Impact of imputing missing data on the estimates of the total daily number of steps:

  1. About 14% of new values are set, all of them with the same value of the all mean, so the result is that we drive both the mean and median to values closer to the old mean
  2. This drove both the median and the mean to the same value.
